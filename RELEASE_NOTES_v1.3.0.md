# macOS Cleaner v1.3.0

这是当前推荐发布版本。

下载：

- 普通用户请直接下载 `macOS Cleaner-macOS.zip`
- 解压后双击 `macOS Cleaner.app` 即可使用

本次更新：

- 新增原生菜单栏状态面板
- 菜单栏支持健康度、磁盘、内存、电池、CPU、网络、网速测试
- 菜单栏支持打开主窗口、打开仪表盘、一键清理缓存、一键释放可回收内存
- 修复菜单栏点击桌面不关闭的问题
- 修复菜单栏内存读数与仪表盘口径不一致的问题
- 修复“隐藏到后台”在部分环境中误伤其他应用的问题
- 修复侧边栏导航在部分设备上字体被上下裁切的兼容性问题

包含：

- 全面检查
- 仪表盘
- 文件清理
- 图片管理
- 磁盘空间
- 软件缓存
- 开机启动
- 应用程序
- 内存管理
- 设置
- 菜单栏状态面板

说明：

- 第一次打开如果被 macOS 拦截，请右键应用后选择“打开”
- 这是未 notarize 的开发版应用
- 如果你机器上有多个 `macOS Cleaner.app` 副本，请确认实际启动的是当前这份发布包
